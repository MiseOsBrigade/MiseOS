# Glossary

- **CAIP scope:** Chain-agnostic identifier for a blockchain namespace and network.
- **Deterministic builder:** Trusted code that produces the same transaction request from the same validated input.
- **Intent:** Expiring, typed description of a proposed blockchain action.
- **Policy decision:** Machine-readable authorization result separate from signature validity.
- **Provenance manifest:** Structured record linking an artifact hash, metadata hash, issuer, storage URI, and external identifiers.
- **Simulation:** Read-only prediction of transaction behavior before signing.
- **Settlement evidence:** Verified Stripe or on-chain record used by the entitlement service.
- **Wallet authentication:** Proof of account control for an application session; separate from transaction authorization.
