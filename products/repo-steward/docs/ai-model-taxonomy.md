# AI model taxonomy

MiseOS uses independent classification lenses.

## Capability
- ANI: current task-bounded systems
- AGI: theoretical human-level general capability
- ASI: hypothetical beyond-human general capability

## Functional depth
- Reactive Machines
- Limited Memory
- Theory of Mind (experimental)
- Self-Aware AI (conceptual)

## Paradigms
Predictive, Generative, Agentic, Multimodal, Causal, Neurosymbolic.

## Learning
Supervised, Unsupervised, Self-supervised, Semi-supervised, Reinforcement Learning, RLHF.

## Architectures
Transformers, CNNs, GANs, diffusion, reasoning models, SLMs, VLMs, MoE, RAG, planning systems, and multi-agent systems.
