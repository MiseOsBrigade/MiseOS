# Frozen roadmap

These initiatives remain valid but are intentionally excluded from Release 0 until Pull Request Guardian works reliably in external repositories:

- Chef Brigade expansion beyond launch agents
- NFT minting and metadata systems
- blockchain attestations
- marketplace payments
- token economy
- 3D GridOps world
- comics and animation platform
- community quests
- multi-cloud execution
- enterprise billing
- public workflow marketplace
- autonomous agent negotiation
- advanced vector memory
- knowledge graph
- mobile application

## Reopening rule

A frozen initiative may re-enter active planning only after:

1. an external developer installs MiseOS without manual intervention;
2. a real pull request triggers Pull Request Guardian;
3. every step produces structured evidence;
4. GitHub receives a useful branded check result;
5. GridOps explains pass, failure, or block state;
6. onboarding succeeds across three external repositories.
