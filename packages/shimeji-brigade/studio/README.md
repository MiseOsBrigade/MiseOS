# Shimeji Brigade v8 Character Studio

Creator-facing workflow:

1. upload a reference image or PDF page
2. adjust the crop box
3. preview transparent-background cleanup
4. approve a canonical 128×128 master
5. generate `idle`, `walk`, `climb`, `fall`, `drag`, `interact`, and `perch` frames
6. inspect validation errors live
7. export runtime packs
8. export clean Shimeji Mint bridge metadata

The studio does not contain wallet logic, private keys, seed phrases, or mint transaction code.
