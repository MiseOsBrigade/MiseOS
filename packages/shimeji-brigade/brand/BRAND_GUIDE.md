# Shimeji Brigade brand guide

## Brand idea

Shimeji Brigade turns reference art into validated autonomous sprite agents.

## Primary tagline

**Tiny agents. Big motion.**

## Voice

Use direct creator-facing language:

- reference-backed
- validated sprite pack
- runtime-ready
- creator studio
- autonomous character
- preview before publishing

Avoid wallet-first or investment language. The Brigade package does not mint, custody keys, or promise collectible value.

## Visual principles

- chibi/kawaii proportions
- thick readable silhouette
- strong role prop
- clean transparent sprite edges
- black/gold MiseOS system language
- cyan/orange accent language for Mizu and Kurogami launch assets
